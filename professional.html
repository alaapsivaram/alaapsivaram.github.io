AI Customer Service

Course: Cloud Computing and Big Data
Major: Computer Science

AI Customer Service Project
  • Added features to a basic iOS app to make it an AI Customer Service chatbot that included facial verification to carry out conversations for either booking a cab, booking a hotel or booking a parking
  • While signing up the user have to pick his username and password and also take a photo which would be stored in the AWS S3 bucket against a unique ID. A corresponding record was entered in the AWS DynamoDB containing the username and the unique ID.
  • During log in, the username and password would be used by AWS Cognito for authentication, but the app would also take a photo of the user. This would trigger a Lambda function, which would extract the corresponding unique ID for the user, and then retireve the image from the S3 bucket that was stored with this ID. AWS Rekognition would then compare the two images and authenticate the user if the similarity was greater than 90%.
  • Once the user was authenticated, the user could get the chatbot to book either a hotel, book a cab or book a parking. This was implemented using AWS Lex having 3 intents (booking cab, booking hotel or booking parking spot) and each having a different set of questions beore the booking was finally confirmed via email confirmation. This was achieved using AWS SES (Simple Email Service).
  • Used AWS services like Rekognition (facial verification), Lex (chat layer), Cognito (sign up/login), API Gateway, Lambda, Kinesis, DynamoDB, S3, Elastic Beanstalk
